# BackgroundSwitch

Всем привет, это программа для смены обоев по нажатии комбинации клавиш. Программа имеет в себе функцию применять фильтр на обои. В целом программа появилась на свет из-за того, что мне просто хотелось случайной смены обои по нажатии комбинации клавиш. 
(кстати программа работает только на винде)

# Гайд по сборке и запуска

Это для тех кто не умеет собирать зависимости проекта через `requirements.txt` и запускать внутри виртуального пространства.

## Создание виртуального пространства

```bash
python -m venv venv # Создание виртуального пространства
venv\Scripts\activate.bat # Вход в пространство
```

## Сборка зависимостей

``` sh
pip install -r requirements.txt # Установка необходимых библиотек
```

## Запуск

``` sh
python main.py
```
